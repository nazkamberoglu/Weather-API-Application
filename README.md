This project is a web application developed using a Weather API.
Current weather information for the selected city is fetched in real time from the API and displayed to the user.

The application provides live data such as temperature, feels-like temperature, humidity, wind speed, and weather conditions.

Features

Weather API Integration

Fetch current weather data by city

Real-time data usage

City-Based Weather Information

Display weather details for the selected city

Temperature, humidity, wind speed, and weather description

Live Data Display

Current and feels-like temperature

Humidity and wind speed information

User Experience

Simple and user-friendly interface

Dynamic data updates
